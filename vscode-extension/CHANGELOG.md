# Change Log — UiAudit VS Code Extension

All notable changes to the UiAudit VS Code Extension will be documented in this file.

## [1.0.0] - 2026-07-20

### Added
- **Real-Time Inline Diagnostics:** Underline accessibility (a11y), performance, and SEO issues on file open and save.
- **Quick Fix Code Actions:** Instant inline recommendations and fix snippets for flagged rules.
- **Command Palette Commands:** Added `UiAudit: Scan Active File` and `UiAudit: Clear Diagnostics`.
- **Extension Settings:** Configurable `uiaudit.enable`, `uiaudit.runOnSave`, and `uiaudit.categories`.
- **Local Config Support:** Automatic loading and enforcement of project `uiaudit.config.json` and `.uiauditrc` files.
